# Text_Encryption
A Text Encryption project developed using Python for backend logic and HTML/CSS for the user interface. Created as part of my Pinnaclelab Cybersecurity Internship.

🔐 ShadowCrypt – Text Encryption System

📌 Project Overview:-

ShadowCrypt is a simple and attractive Text Encryption & Decryption System built using:

🐍 Python

🌐 HTML

🎨 CSS

🔢 Caesar Cipher Algorithm:-

This project allows users to securely encrypt and decrypt secret messages using a custom encryption key.

🔒 Encrypt Secret Messages

🔓 Decrypt Encrypted Messages

📱 Responsive Interface

👨‍💻 Beginner-Friendly Project

⚡ Fast & Lightweight

🧪 How It Works


The system performs:
1️⃣ User enters:-
Message

Encryption key
2️⃣ Python generates encrypted output using Caesar Cipher:
🔢 Formula:-
E(x) = (x + n) mod 26

Where:

x = original character

n = shift/encryption key

26 = total alphabets

▶️ How to Run

(Shift of +3 applied)
